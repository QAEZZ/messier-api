messier-api

a simple API for getting information on objects in the messier catalog


..... Base URL GitHub
Base URL
https://messier-api.russiandev.repl.co/messier

Type URL Parameters Description
GET ?object= object number: int Returns data on a specified object
GET /random none Returns data on a random object

Example response from https://messier-api.russiandev.repl.co/messier?object=104
